# awesome-android
A curated list of awesome Android libraries and projects.

### Network
[okhttp-OkGo](https://github.com/jeasonlzy/okhttp-OkGo) -🔥全新完美支持RxJava，比Retrofit更简单易用

[OKHttpUtils](https://github.com/duzechao/OKHttpUtils) -对OkHttp进行封装，网络请求失败自动查询本地缓存,上传文件等功能

[Lina-RxOkHttp](https://github.com/pinguo-sunjianfei/Lina-RxOkHttp) -基于RxJava封装的OkHttp网络请求库

[thunder](https://github.com/easybenefit/thunder) -一种基于编译期注解的Android网络请求框架

[RetrofitOKHttpCache](https://github.com/maning0303/RetrofitOKHttpCache) -测试Demo，使用Retrofit和Okhttp实现网络缓存。无网读缓存，有网根据过期时间重新请求

[AndNet](https://github.com/qibin0506/AndNet)

[DaVinci](https://github.com/CPPAlien/DaVinci) -基于Volley实现的一款多功能网络库, 目前包括了普通图片和Gif图片的加载显示、图片的二级缓存机制、可以开启Cookie的http请求等功能

[RetrofitUtils](https://github.com/StudyLifeTime/RetrofitUtils) -Retrofit+Okhttp3+Rxjava的网络请求库

[GoHttp](https://github.com/xiaopansky/GoHttp) -GoHttp是Android上的一个用来发送Http请求的库，旨在用最简单、最快捷的方式从网络上拿到我们想要的数据

[KakaCache](https://github.com/LittleFriendsGroup/KakaCache) -咔咔一声，缓存搞定。这是一个专用于解决Android中网络请求及图片加载的缓存处理框架

[KakaCache-RxJava](https://github.com/LittleFriendsGroup/KakaCache-RxJava)

[RestHttp](https://github.com/llxdaxia/RestHttp) -Android简单易用的网络库，支持 Http，Https，Http 文件上传，图片加载，面向接口的 API 调用方式，轻量的设计

[HttpUtilForAndroid](https://github.com/hss01248/HttpUtilForAndroid) -深度结合android平台特性的网络封装库,api链式调用,集成cookie管理,多种缓存模式,极简https配置,上传下载进度显示,以及丰富的下载后处理功能,实现了自动登录成功后的请求重发功能,多种形式的参数设置和中国特色的json解析.使用时基本是操作boolean,int,string,不涉及到具体第三方库中的类.下层默认是okhttp实现,可无缝切换到其他网络库

[HttpRequest](https://github.com/leibing8912/HttpRequest) -一个基于 retrofit + okhttp + Gson 封装的网络框架

[Fast-Android-Networking](https://github.com/amitshekhariitbhu/Fast-Android-Networking) -🚀 A Complete Fast Android Networking Library that also supports HTTP/2 🚀

[HttpRequest](https://github.com/robinxdroid/HttpRequest)

[ReactiveNetwork](https://github.com/pwittchen/ReactiveNetwork) -Android library listening network connection state and Internet connectivity with RxJava Observables

[okhttp](https://github.com/square/okhttp) -An HTTP+HTTP/2 client for Android and Java applications

[android-async-http](https://github.com/loopj/android-async-http) -An Asynchronous HTTP Library for Android
